# 分岐型小説シナリオ：ブルーシートのゆりかご

**版**: v1.0（ゲーム実装反映）

---

## 0. この物語の法則（構成の芯）

我孫子武丸的な分岐設計——**正解はない。あるのは、限られた一枚を誰のために使うかという選択だけ。**

| 段階 | 問い | S01例 | 読者に試されるもの |
|------|------|-------|-------------------|
| 第1分岐 | **誰の側に立つか** | 遥／配給／（隠）木下爺 | 弱者優先・秩序・知恵 |
| 第2分岐 | **一枚を何に使うか** | ゆりかご／抱っこ／配給堅持 | 即効性・公平・保留 |
| 第3分岐 | **留まるか戻るか** | 避難所／父の家／（隠）子守歌 | 他者と自分の父 |

**走るモチーフ**（各シーンに散らす）:
- 体育館の天井：**雨粒が落ちる音**（ブルーシートが必要な場所）
- 蓮の泣き声：**三時間前から止まっていない**——避難所全体の神経を削る
- 父からの未読：**「大丈夫、こっちは平気」**——最後の一行だけが嘘くさい

**語り口**: 一人称・現在形。保育士としての観察眼と、娘としての罪悪感が交互に顔を出す。情景は淡々と、感情は一行で刺す。

## 1. 作品概要（100字以内）

**フック**: 震災三日目、体育館の避難所。生後二ヶ月の蓮が三時間泣き続け、誰も眠れない。届いたブルーシートは六枚だけ——屋根の雨漏れ、寝床、足元の泥。大工だった木下爺が言う。「一枚あれば、ゆりかごが作れる」。父の家は、二キロ先で音が途切れている。

**選択の意味**: 一枚のシートを誰のために使うか。留まるか、戻るか。正解はなく、残るのは「その夜、あなたがどんな人間だったか」だけ。

## 3. 分岐マップ（v1.0）

**三つの問いで読むマップ** — ①誰の側か → ②一枚の使い道 → ③留まるか戻るか

```
START
└─ S01（分岐①）
   ├─ [A] 遥のところへ → S02-A
   ├─ [B] 配給係を手伝う → S02-B
   └─ [隠・2周目] 木下爺に声をかける → S02-C
       │
S02-A ─┬─ [A] シートをゆりかごに → S03-CRADLE
       └─ [B] まず抱き上げる → S03-HOLD
       │
S02-B ─┬─ [A] 蓮を優先する → S03-CRADLE
       └─ [B] 配給ルールを守る → S03-RULE
       │
S02-C（自動）→ S03-CRADLE（木下流・carpenter_method）
       │
S03-CRADLE ─┬─ [A] 夜明けまで残る → E01
            ├─ [B] 父の家へ → E03
            └─ [隠・全ED回収] 子守歌を流す → E05
       │
S03-HOLD ─┬─ [A] 夜明けまで残る → E02
          └─ [B] 父の家へ → E04
       │
S03-RULE ─┬─ [A] 夜明けまで残る → E02
          └─ [B] 父の家へ → E04
```

### 隠しルート（Secret: E05）

**解放条件**: E01・E02・E03・E04 をすべて回収済み

1. 2周目で S01「木下爺に声をかける」→ S02-C（`carpenter_method`）
2. S03-CRADLE で「遥の子守歌を流す」→ E05

## 4. シーン一覧

---

**[シーンID]**: S01  
**章題**: 三時間、止まらない  
**種別**: 分岐点  
**前シーン**: START  
**想定文字数**: 580字

**情景描写**: 市立第三中学校、体育館。震災三日目の夜八時。懐中電灯と非常灯だけ。天井の端から、一定の間隔で水が落ちる音。

**本文**:

体育館の入り口で、靴の泥を払う。避難者三百四十人。段ボールの寝床が並び、通路には荷物と人の足が混ざっている。

奥から、赤ん坊の泣き声が来る。三時間前から、一度も長く止まっていない。誰かがため息をつく。誰かが布団を被る。泣き声は壁を越え、神経をなぞるように削っていく——保育士の私には、その意味がわかりすぎて苦しい。

配給所のテント前では、今日届いた救援物資の開封作業が始まっている。段ボール箱に**ブルーシート**の文字。担当の職員が言う。「六枚。屋根の雨漏れ、足元、寝床——全部で足りない」

スマホを見る。父からのLINEは、昨日の午前中で止まっている。

**「大丈夫。こっちは平気。お前は行ってこい」**

既読はついている。返信はない。二キロ先の実家が、地図上では近いのに遠い。

隅の柱の陰に、杖をついた老人が座っている。木下さん。避難初日から、ほとんど喋らない。膝の上に、古い作業手袋を置いている。

遥が抱っこ紐で蓮を抱え、配給列の端に立っている。目の下にクマ。唇が乾いている。

——足を向ける先を、決めなければならない。

**選択肢**:
- A: 「遥のところへ行く」 → S02-A
- B: 「配給係を手伝う」 → S02-B
- 隠: 「木下爺に声をかける」 → S02-C（2周目）

---

**[シーンID]**: S02-A  
**章題**: 一枚の行方  
**種別**: 分岐点  
**前シーン**: S01-A  
**想定文字数**: 520字

**本文**:

遥の前にしゃがむ。蓮の顔は真っ赤で、小さな拳を握りしめている。授乳は十分——それでも泣く。触覚過敏、環境の変化、単に疲れている。理由はいくつもあるが、今は一つも通じない。

「三時間……」遥の声が掠れる。「ごめん、みんなに迷惑かけて。でも、止まらなくて」

「迷惑じゃない。——一枚、使っていい？」

配給所のブルーシート。六枚のうち一枚を、ゆりかごにする案を頭に浮かべる。体育館の隅、雨が落ちている高齢者コーナーにも、同じシートが必要だ。

蓮を抱き上げる。二キログラム弱。軽い。それなのに、腕の中で震えが伝わってくる。保育士として、まず触れ合いを試すべきか。それとも、すぐに環境を変えるべきか。

**選択肢**:
- A: 「シートをゆりかごに」 → S03-CRADLE
- B: 「まず抱き上げる」 → S03-HOLD

---

**[シーンID]**: S02-B  
**章題**: 六枚の配分表  
**種別**: 分岐点  
**前シーン**: S01-B  
**想定文字数**: 500字

**本文**:

配給所のテーブル。職員の田村さんが配分表を広げている。

「屋根三箇所、足元二列、寝床用に最低一枚——で、六枚全部」

列に並ぶ人々。高齢者、子連れ、単独避難者。遥と蓮は、列の後ろで身を揺らしている。泣き声が、配分表の数字より先に耳に入る。

田村さんが小声で言う。「あの赤ちゃん、さっき区の職員が『優先枠』って言ってた。でも、ルール上は——」

「優先枠？」

「要配慮者向けの物資。判断は現場任せ、と」

蓮の泣き声が、また一段大きくなる。誰かが「うるさい」と呟く。遥の肩が落ちる。

公平に配るか。今この場で、一枚を蓮のために動かすか。

**選択肢**:
- A: 「蓮を優先する」 → S03-CRADLE
- B: 「配給ルールを守る」 → S03-RULE

---

**[シーンID]**: S02-C  
**章題**: 大工の手  
**種別**: 本編（自動進行）  
**前シーン**: S01-隠  
**想定文字数**: 480字

**本文**:

木下爺の前に座る。彼は顔を上げ、初めてこちらを見た。

「……何か用かね」

「ゆりかご、作れますか。あの赤ちゃんの」

長い沈黙。彼は膝の上の手袋をはめ、ゆっくり立ち上がる。

「一枚あれば、足りる。麻縄があれば、もっと早い」

柱に近づき、天井の梁を見上げる。非常灯の赤い光が、皺の深い顔を照らす。

「若い頃、孫が生まれた年、台風で店が潰れた。同じことをした。ブルーシートと、倉庫の麻縄でな」

指先が空中で結び目を描く。蝶結び、縮み結び、荷重の掛け方——言葉より早い。

「お前、保育士だろう。赤ん坊の泣き声は、みんなの泣き声だ。——シートを持ってこい。結び方は、俺が教える」

**自動進行**: 「配給所へ向かう」→ S03-CRADLE

**[シーンID]**: S03-CRADLE  
**章題**: 吊るされた青  
**種別**: 分岐点  
**前シーン**: S02-A / S02-B / S02-C  
**想定文字数**: 550字

**本文**:

ブルーシートを梁に架ける。木下流の結び方なら、三分で形になる。シートの中央がうねり、浅い舟のように垂れる。蓮を載せると——

体重がかかり、シートが静かに揺れる。

蓮の泣き声が、小さく途切れた。一瞬。二瞬。そして、再び大きく——だが、揺れに合わせて、声のトーンが変わった。怒鳴りではなく、困惑に近い音。

遥が息を呑む。「……動いてる。ゆっくり、動いてる」

体育館のざわめきの中で、小さな揺れだけが、別の時間を作っている。

スマホが震える。父から——いや、着信なし。通知もない。

時計は夜十時。あと八時間、避難所の当番が続く。父の実家までは、自転車で二十分。今なら、行ける。

**選択肢**:
- A: 「夜明けまで残る」 → E01
- B: 「父の家を見に行く」 → E03
- 隠: 「遥の子守歌を流す」 → E05（全ED回収＋木下流）

**[シーンID]**: S03-HOLD  
**章題**: 腕の限界  
**種別**: 分岐点  
**前シーン**: S02-A-B  
**想定文字数**: 480字

**本文**:

蓮を抱き上げ、肩で軽く揺らす。保育士として何百回とやってきた動作。だが、避難所の騒音の中では、効果が薄い。

三十分。腕が痺れる。遥と交代する。彼女の抱っこ紐に移し替え、また泣き声が大きくなる。

「ごめん……私、ダメみたい」

シートは配給所に残っている。雨漏れの音は、高齢者コーナーから続いている。

父のLINE。未読のまま。二キロ先。

**[シーンID]**: S03-RULE  
**章題**: 配分表どおり  
**種別**: 分岐点  
**前シーン**: S02-B-B  
**想定文字数**: 460字

**本文**:

田村さんの配分表どおり、六枚を配った。屋根三箇所、足元二列、寝床一枚。

蓮は、まだ泣いている。遥は列の端で、小さくうなずいただけ。「わかってる。——ありがとう、配給の件」

それ以上、何も言えない。ルールを守った。それは正しいはずだ。

高齢者コーナーの雨漏れは止まった。代わりに、赤ん坊の泣き声が体育館全体を満たしている。
