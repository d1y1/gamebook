# 分岐型小説シナリオ：隣人の手紙

**版**: v1.0（ゲーム実装反映）

---

## 0. この物語の法則（構成の芯）

我孫子武丸的な分岐設計——**犯人探しではない。加害の輪の中に、読者自身を置く。**

| 段階 | 問い | 例 | 読者に試されるもの |
|------|------|-----|-------------------|
| 第1分岐 | **知るか、届けるか** | 封筒を調べる／急いで届ける | 好奇心と義務感 |
| 第2分岐 | **封を破るか** | 読まず届ける／中身を見る | 約束と真実 |
| 第3分岐 | **受け入れるか、否認するか** | 謝る／「関係ない」 | 自分の加害性 |

**走るモチーフ**（各シーンに散らす）:
- 壁の向こう：配信の笑い声、インスタントの匂い、**二度だけのノック**
- 三通の封筒：花のステッカー／会社ビル／新潟の年賀状の記憶
- 柏木の一行：**「あなただけが、残っていたから頼む」**

**語り口**: 一人称・過去形と現在形の混在。地の文は短く切る。説明より観察。感傷は一行で済ませ、すぐ次の事実（匂い、封筒の厚み、チャイム）に移る。

## 1. 作品概要（100字以内）

**フック**: 孤独死した隣人から、三通の手紙が届く。届けてほしいと。名前も素性もわからない宛先を、推理していくうちに——あなたも、手紙の宛先の一人だった。

**選択の意味**: 配達人になるか、共犯を認めるか。開けるか、焼くか。正解はなく、残るのは「隣人だったかどうか」だけ。

## 3. 分岐マップ（v1.0）

**三つの問いで読むマップ** — ①知るか届けるか → ②封を破るか → ③罪を認めるか

```
START
└─ S01（自動）→ S02
   └─ S02（分岐①）
      ├─ [A] 封筒の表書きを調べる → S03
      └─ [B] すぐ届けに行く → S03-RUSH → S07
         │
S03 ─┬─ [A] 花のステッカー（A） → S04-A ─┐
     ├─ [B] 会社ビル（B） → S04-B ─────┼→ S05
     └─ [C] 新潟（C） → S04-C ─────────┘
       │
S05 ─┬─ [A] 読まず届ける → S07 → S08
     ├─ [B] 封を開ける → S06-OPEN ─┬→ S07
     │                              └→ S08
     └─ [隠・2周目] ノックを思い出す → S05-MEM → S07
       │
S08 ─┬─ [A] インターホンを見る → S09
     └─ [B] まだ開けない → S09-WAIT → S09
       │
S09 ─┬─ [A] 母親を通し手紙を受け取る → S10
     ├─ [B] 手紙Cを母親に渡す → S10-GIVE → S10
     └─ [隠・ノック回想済] 告白する → S10-CONFESS → S10
       │
S10 ─┬─ [A] 母親に謝る → S11-A ─┬→ E01（届ける）
     │                           ├→ E02（焼く）
     │                           └→ [隠・告白済] E05（手紙に返事）
     ├─ [B] 「関係ない」 → S11-B → E03
     └─ [C] 残りを届ける → S11-C → E04
```

### 隠しルート（Secret: E05）

1. **2周目**で S05 に「あの夜、ノックの音を思い出す」が出現（`remembered_knock`）
2. S09 で「あの夜、ノックがありました」と告白（`confessed_knock`）
3. S11-A で「あの夜のことを、手紙に書いて返す」→ E05

## 6. 執筆メモ

### 主要ロケーション
1. 分譲マンション二〇三号室（主人公）・二〇四号（柏木誠）
2. 花森ゆい所属事務所（渋谷）
3. ライトウェイ・ソリューションズ（区内オフィスビル十五階）
4. 柏木節子の実家（新潟県長岡市）——物語上は訪問せず

### 共通描写（再利用可）

1. **モチーフ①**: 壁越しの配信の笑い声／インスタントの匂い
2. **モチーフ②**: 二回だけのノック（Secretの鍵）
3. **モチーフ③**: 三通の封筒の質感（ステッカー／ビル名／厚い便箋）
4. **モチーフ④**: 柏木の一行「あなただけが、残っていたから頼む」
5. **チャイム**: 母親登場の合図——読者も「開けるか」問われる

### 選択肢表示（Web / アプリ向け）
- 分岐点末尾はシーンごとに異なる問い（「どの手がかりから」「この手紙を、どう受け止める？」）
- 隠し選択肢は2周目表示、または `remembered_knock` フラグ必須
- Secret（E05）は初回到達率5%程度。ヒントはタイトル画面の「あの夜のノック」
- **本文にフラグ名・制作メモを入れない**（実装は `scenes.js` 側）
