# 分岐型小説シナリオ：完全なる殺人

**版**: v1.0（ゲーム実装反映）

---

## 0. この物語の法則（構成の芯）

我孫子武丸的な分岐設計——**犯人探しが正解ではない。読者が「犯人」を選ぶたび、別の犯人が生まれる。**

| 段階 | 問い | 例 | 読者に試されるもの |
|------|------|-----|-------------------|
| 第1分岐 | **何から手をつけるか** | 物証／人間関係 | 捜査の切り口 |
| 第2分岐 | **どの容疑者を追うか** | 前妻／同僚／秘書／弟 | 動機と物証の結びつき |
| 第3分岐 | **確信するか、掘るか** | 即起訴／追加調査 | 組織の圧力と直感 |

**走るモチーフ**（各シーンに散らす）:
- 鍵のかかった書斎：内側から施錠、窓も施錠——密室に見える
- ブランデイのグラス：二つ並ぶ。片方に口紅
- 四人分の物証：イヤリング／カフリンク／口紅のグラス／切れたベルト
- ホワイトボードの名前：藤沢・神谷・水城・佐々木

**語り口**: 一人称・過去形。補佐刑事の観察眼。鑑識用語は必要最小限。感情は一行で切る。

**真相の扱い（重要）**:
- 本編ルート（E01〜E04）では、いずれも「他殺・犯人逮捕」で完結する
- 自殺と証拠の仕込みは、**E05（Secret）でのみ明示**
- E01〜E04 回収後に S02 へ「鑑識の未公開メモ」が出現

## 1. 作品概要（100字以内）

**フック**: 鍵のかかった書斎で、不動産王の死体が見つかる。現場には四人分の恨みが——イヤリング、カフリンク、口紅、切れたベルト。あなたは、どの容疑者を追う？

**選択の意味**: 証拠は四人ともに繋がる。選んだ容疑者が「犯人」になる——それが、この事件の仕組みだった。

## 3. 分岐マップ（v1.0）

**三つの問いで読むマップ** — ①物証か人間関係か → ②どの容疑者か → ③確信するか掘るか

```
START
└─ S01（自動）→ S02
   └─ S02（分岐①）
      ├─ [A] 現場の物証 → S03-PHYS
      ├─ [B] 人間関係の糸 → S03-REL
      └─ [隠・全ED回収] 鑑識の未公開メモ → S-SECRET-01 → S-SECRET-02 → E05
         │
S03-PHYS ─┬─ [A] イヤリング（藤沢） → S04-WIFE
          └─ [B] カフリンク（神谷） → S04-PARTNER
         │
S03-REL ──┬─ [A] 脅迫メール（水城） → S04-SECRETARY
          └─ [B] 遺産争い（佐々木） → S04-BROTHER
         │
S04-WIFE ─┬─ [A] 藤沢を追う → E01
          └─ [B] 現場を見直す → S05-WIFE → E01
S04-PARTNER ┬─ [A] 神谷を追う → E02
            └─ [B] 財務を掘る → S05-PARTNER → E02
S04-SECRETARY ┬─ [A] 水城を追う → E03
              └─ [B] 送信経路を追う → S05-SECRETARY → E03
S04-BROTHER ┬─ [A] 佐々木を追う → E04
            └─ [B] 遺言書を調べる → S05-BROTHER → E04
```

### 隠しルート（Secret: E05）

**解放条件**: E01・E02・E03・E04 をすべて回収済み（`secretEndingIds` 以外の全エンディング）

1. 上記条件を満たした状態で S02 に「鑑識の未公開メモ」が出現
2. S-SECRET-01 で鑑識内部の所見（刺入角度・鎮静剤・刃物の所在）を読む
3. S-SECRET-02 で白石隆史の遺書草稿（四人への仕込み手順）を読む → E05

## 5. 真相（Secret のみで明示）

白石隆史は**自殺**した。恨みを持つ四人に疑いを向けさせるため、死後に見えるよう証拠を配置した。

| 証拠 | 読者が信じる解釈（E01〜E04） | 真実（E05） |
|------|------------------------------|-------------|
| イヤリング | 藤沢が現場で落とした | 調停日に拾い、半年眠らせてから配置 |
| カフリンク | 神谷が犯行に関与 | 飲みの席でジャケットを貸し、カフリンクを外させた |
| 口紅のグラス | 水城が訪問した | 同じ色の口紅を白石が購入し、グラスに付けた |
| 脅迫メール草稿 | 水城の言葉を白石が記録 | 白石が彼女の言い回しを真似て自ら書いた |
| ベルト | 佐々木が現場にいた | 昼食後に置いていったベルトを、書斎に配置 |
| 刺し傷 | 他殺 | 鎮静剤服用後、自刃。角度は自刃パターン |
| 密室 | 犯人が鍵をかけた | 内側から自分で施錠 |

遺書草稿の核心：**「完全なる殺人——殺人に見える死を、私は演出する」**

## 6. 執筆メモ

### 主要ロケーション

- 白石邸書斎（二階・港眺望）
- 捜査本部・会議室
- 各容疑者の住居／オフィス
- 鑑識課・押収品室（Secret のみ）

### 共通描写（再利用可）

1. **書斎の朝**: ブランデイの甘い匂い、赤い革張りデスク、港の白い霞
2. **物証袋**: プラスチックの音、係長の付箋、会議室の蛍光灯
3. **坂道**: 白石邸への急坂、看板の色あせ、新しい住人のカーテン（E05）

### 本編で避ける表現

- 刺入角度の不自然さへの言及
- 「証拠が仕込まれている」示唆
- 鎮静剤・自刃の直接的言及
- 遺書・自殺草稿の存在

### リプレイ設計

- E01〜E04 はいずれも「犯人逮捕」で満足できる結末
- 各エンディング末尾に、他ルートへの軽い誘導（「他の容疑者のルートが、まだ残っている」）
- Secret は「四人とも犯人にできた」こと自体が到達条件
