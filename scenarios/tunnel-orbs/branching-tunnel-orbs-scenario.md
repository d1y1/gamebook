# 分岐型小説シナリオ：幽霊に嫌われた話

**版**: v1.0（ゲーム実装反映）

---

## 0. この物語の法則（構成の芯）

短編ホラーの分岐設計——**正解はない。ただ「吸い込まれる側」に回るかどうか。**

| 段階 | 問い | 例 | 読者に試されるもの |
|------|------|-----|-------------------|
| 第1分岐 | **行くか、止めるか** | その夜行く／やめようと言う | 同調圧力と直感 |
| 第2分岐 | **撮るか、拒否するか** | 扉の前に立つ／やめようと提案 | 恐怖への屈服 |
| 第3分岐 | **助けを求めるか** | 連絡する／我慢する | 孤立のリスク |
| 第4分岐 | **ドアの前で** | 待つ／開ける／返事する | 未知への接触 |

**走るモチーフ**:
- オーブのない一枚：友達全員に写り、自分だけ写らない
- ラップ音 → 足音 → 「ダン」：段階的エスカレーション
- 母の一言：**「全部吸い込んでるよ」**

**語り口**: 一人称・過去形。短大時代の回想。強がりと本音の落差。

## 1. 作品概要（100字以内）

**フック**: 心霊スポットで撮った写真——友達にはオーブが写っていた。自分だけ、一枚も。笑い合ったその夜から、部屋でラップ音が鳴り始めた。

**選択の意味**: 行くか、撮るか、一人で耐えるか。オーブは「嫌われた証」ではなく「集められた証」かもしれない。

## 3. 分岐マップ（v1.0）

```
START
└─ S01
   ├─ [A] その夜のうちに行く → S03
   └─ [B] やめておこうと言う → S02-B → S03
      │
S03（自動）→ S04（自動）→ S05（分岐②）
   ├─ [A] 勇気を出して前に立つ → S05
   ├─ [B] やめようと提案 → S04-STOP → S05
   └─ [隠・全ED回収] 母の言葉を思い出す → S04-MEM
      ├─ 撮影拒否 → E05（Secret）
      └─ 最後の一枚 → S05（remembered_mother）
      │
S05 → S06 → S07（分岐③）
   ├─ [A] 友達に連絡 → S08
   └─ [B] 我慢して眠る → E02（Bad）
      │
S08（分岐④）
   ├─ [A] 静かに待つ → S09
   ├─ [B] ドアを開ける → S08-OPEN → S09
   └─ [隠・remembered_mother] ラップ音に返事 → S08-REPLY → S09
      │
S09
   ├─ [A] コンビニで待つ → S10
   └─ [B] 友達について外へ → E03（Normal）
      │
S10
   ├─ [A] 母に電話 → S11 → E01（Bad・正史）
   └─ [B] 一人で部屋に入る → E04（Bad）
```

### 隠しルート（Secret: E05）

**解放条件**: E01・E02・E03・E04 をすべて回収済み

1. S04 に「母の言葉を思い出す」が出現
2. 「撮影を拒否して帰る」を選択 → E05

## 5. 原作との対応

本作は、短大時代にバイト友達と心霊スポット（廃トンネル・封鎖扉）へ行き、インスタント写真でオーブの有無が分かれた体験談をベースに、分岐型小説として再構成したものです。

正史ルート（E01）では、ラップ音・麻雀の夜・「ダン」・コンビニ・母の電話「全部吸い込んでるよ」まで、原作の流れを踏襲しています。

## 6. 執筆メモ

**主要ロケーション**
- 廃トンネル（横並びで歩く）
- 封鎖された鉄扉（インスタント撮影）
- 一人暮らし角部屋（T字路廊下）
- 明け方のコンビニ

**ゲーム実装**
- `storageKey`: `tunnel-orbs-v1`
- 隠し選択肢: `requireAllEndingsExceptSecret`（S04）、`requireFlag: remembered_mother`（S08）
