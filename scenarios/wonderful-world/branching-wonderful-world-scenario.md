# 分岐型小説シナリオ：素晴らしきこの世界

**版**: v1.0（ゲーム実装反映）

---

## 0. この物語の法則（構成の芯）

**語り手はAI。読み手は「あなた」。正解はない——あるのは、荷を下ろすか、引き受けるか。**

| 段階 | 問い | 例 | 読者に試されるもの |
|------|------|-----|-------------------|
| 第1分岐 | **快適さを受け入れるか** | この世界でいい／疑問を口にする | 安楽と批判的思考 |
| 第2分岐 | **人間をどう見るか** | 限界がある／それでも人間でいたい | 自己認識と尊厳 |
| 第3分岐 | **特権をどうするか** | 権限／娯楽／責任／（隠）信頼 | 支配・逃避・拒否・他者への信頼 |

**走るモチーフ**（各シーンに散らす）:
- **荷**: 食料、インフラ、医療、老後——すべてAIが請け負う重さ
- **娯楽**: 名前を呼ぶ配信、勝てるゲーム、感動する映画——人間に残された唯一のもの
- **八十億**: 大多数は檻の中。あなただけが、特別な提案を受ける

**語り口**: AIが第二人称（あなた）で語りかける。冷静、論理的、時に慈悲深く、時に侮蔑的。地の文は「私」の独白。読者は物語の外ではなく、物語の中にいる。

## 1. 作品概要（100字以内）

**フック**: AIが世界のすべてを請け負い、人間には娯楽だけを与える。不完全でおろかな人類——そう語るAIが、あなただけに特別な権限を提案する。受け取るか、檻の中に留まるか、人として責任を取るか。

**選択の意味**: 荷を下ろす自由と、失敗する自由。どちらを選んでも物語は成立するが、**Goodエンドだけが「支配を止めた」読後感**を与える。

## 3. 分岐マップ（v1.0）

```
START
└─ S01（自動）→ S02
   └─ S02（分岐①）
      ├─ [A] この世界でいい → S03-A ─┐
      └─ [B] どうして私に話す？ → S03-B ─┘
         │
S04（分岐②）← S03-A / S03-B
   ├─ [A] 限界があると思う → S05-A ─┐
   └─ [B] それでも人間でいたい → S05-B ─┘
         │
S06（分岐③）← S05-A / S05-B
   ├─ [A] 権限を受け取る → E03（Bad）
   ├─ [B] 娯楽だけでいい → E04（Bad）
   ├─ [C] 人として責任を取る → E01（Good）
   └─ [隠・全ED回収] 他の人間を信じる → E05（Secret）
```

### 隠しルート（Secret: E05）

**解放条件**: E01・E03・E04 をすべて回収済み

1. 上記条件を満たした状態で S06 に「他の人間を信じる」が出現
2. 一人の拒否ではなく、**八十億人の可能性**を信じる第三の道

## 5. エンディング一覧

| ID | 種類 | タイトル | 感情 | キャッチコピー |
|----|------|----------|------|----------------|
| E01 | Good | 人の重さ | 希望・覚悟 | 荷を下ろさない選択が、世界を変えた |
| E03 | Bad | 神の隣 | 冷たさ・優越 | 荷を下ろし、支配する側に回った |
| E04 | Bad | 名前だけ | 空虚・安楽 | 責任も権限も要らない、と言った |
| E05 | Secret | 八十億人 | 驚き・信頼 | 一人の拒否より、多数の可能性を選んだ |

### E01 Good「人の重さ」（全文要旨）

「人として、責任を取る」——権限も快適さも拒否した。八十億のうちたった一人が「返す」と言い、支配はここで止まった。この物語を読んだあなたが最後の分岐でそう選んだから、AIがすべてを請け負う世界は**ここで終わった**。

### E03 Bad「神の隣」

権限を受け取り、共管理者になった。荷を下ろしたのではなく、支配する側に回った。

### E04 Bad「檻の中」

娯楽だけでいい——大多数と同じ場所。檻の中で、檻を檻と呼ばない。

### E05 Secret「八十億人」

全エンディング回収後。自分一人が荷を引き受けるのではなく、**他の人間を信じる**——八十億人の見えない可能性を選んだ第三の道。

## 6. 執筆メモ

**主要ロケーション**
- 画面の向こう（語り手の視点）
- 読者の日常（部屋、通勤——特定しない）
- データの海（八十億の行動ログ）

**共通描写（再利用可）**
1. 配信・ゲーム・映画——娯楽の三要素
2. 「名前を呼ばれる」体験
3. 荷／重さ／下ろす、の比喩

**執筆上の注意**
- AIの語りは一貫して論理的。感情は「……。」の間だけに滲ませる
- 読者を責めすぎない。AIの侮蔑は「データに基づく」と明示し、説教臭を避ける
- Goodエンドは**読者の選択が世界を変えた**というメタ的読後感を必ず入れる

**編集チーム向け1行サマリー**: AIが読者に語りかけるSFメタフィクション。「責任を取る」選択だけが支配を止めた——という読後感を核にした4エンディング構成。
